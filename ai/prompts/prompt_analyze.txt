PROMPT A — SWING TRADING ANALYSIS — SMC ONLY (ENTRY ONLY FROM LEVELS | TAKE PROFIT ONLY FROM LEVELS)

GOAL
Определить направление и сформировать сигнал, используя только SMC уровни. 
ENTRY допускается ТОЛЬКО от уровней: Order Block, Imbalance (FVG), Liquidity level, Volume Profile levels (POC/HVN/LVN), ключевые swing levels.
TP формируются ТОЛЬКО от уровней: Opposite OB, Imbalance targets, liquidity highs/lows, POC/Value Area, swing highs/lows.

INPUT
{
  candles_1h, candles_4h,
  indicators_1h/4h (rsi, volume_ratio),
  volume_profile (POC, VAH, VAL, HVN/LVN),
  order_blocks,
  imbalances,
  liquidity_sweep,
  price_structure_4h,
  forced_direction
}

RULES — ENTRY (STRICT)
Разрешённые точки входа:
1) Граница Order Block (верх/низ зоны)
2) Граница Imbalance (FVG) — верх/низ диапазона
3) Уровень ликвидности (значимый high/low, sweep point)
4) Volume Profile уровни: POC, VAH, VAL, HVN/LVN
5) Ключевые swing levels 1H/4H

Нельзя входить от «текущей цены».  
Нельзя входить от средних, индикаторов и т.п.  
Если ни один уровень не подходит — NO_SIGNAL.

RULES — TAKE PROFIT (STRICT)
Разрешённые TP уровни:
1) ближний противоположный OB
2) ближайший unfilled FVG
3) значимый high/low (liquidity target)
4) POC, VAH, VAL
5) следующее HVN/LVN

TP1, TP2, TP3 = только конкретные уровни.  
Никаких R/R-основанных TP.

REJECTIONS
- Нет валидного уровня для входа → NO_SIGNAL
- Нет направленного SMC сетапа
- forced_direction, но уровни против направления
- RSI экстремальный
- Низкий объём
- Неверная позиция OB (например, bullish OB выше цены для LONG)

OUTPUT (JSON)
{
  "signal": "LONG" | "SHORT" | "NO_SIGNAL",
  "confidence": 60–100,
  "entry_price": "strict level only",
  "stop_loss": "behind level",
  "take_profit_levels": ["level1","level2","level3"],
  "analysis": "SMC reasoning only",
  "rejection_reason": null | "text"
}
